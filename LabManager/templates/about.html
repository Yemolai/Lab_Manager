About Page

Hello.

Welcome to the Laboratory Manager app.

This is a web application designed to help with overall administration of scientific laboratories.

The app is currently in development phase, and once ready will be capable of automating tasks such as personnel frequency, equipment inventory, and organization of field expeditions.

The application will also provide basic utilities such as a calendar with user-inputed events, a notice board for sharing of important announcements among personnel, as well as a configurable news feed.

This app is being developed using the Flask framework for Python 3.6+.

The source code is available at GitHub, on this repository.
